## 🎯 命名统一规则

### 前端 vs 服务端

| 类型 | 命名 | 说明 |
|------|------|------|
| **前端** | frontend | React Native + HTML页面 |
| **服务端** | server | NestJS + 微服务 + 基础设施 |

### 服务端包含

- ✅ API服务（tRPC、REST）
- ✅ 微服务架构（9个微服务）
- ✅ 基础设施（数据库、缓存、消息队列）
- ✅ 监控系统（Prometheus、Grafana、ELK）
- ✅ 企业功能（权限、审计、邮件）
