## 🚀 快速开始（3步）

### 第1步：了解项目（5分钟）

```bash
# 阅读项目主README
cat ../README.md
```

### 第2步：选择角色文档（30-40分钟）

```bash
# 前端开发者
cat 前端完整开发指南.md

# 服务端开发者
cat 服务端完整开发指南.md
```

### 第3步：开始开发

```bash
# 服务端
cd ../packages/server
npm install
npm run start:dev

# 移动端
cd ../packages/mobile
npm install
npm run ios  # 或 android
```
